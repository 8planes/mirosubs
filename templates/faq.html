  1. {% trans 'What is Universal Subtitles?' %}

    {% blocktrans %} Universal Subtitles is an easy way for anyone to add subtitles, captions, or translate nearly any video on the web. It works in a collaborative Wikipedia-like fashion. The software is free and open source and can be used and improved by anyone. {% endblocktrans %}

  2. {% trans 'How can I use Universal Subtitles?' %}

    {% blocktrans %} You're welcome to try our demo to see how the software works. You can also begin subtitling a video you have already posted on the web. {% endblocktrans %}

  3. {% trans 'What are the benefits of subtitles?' %}

    {% trans 'VVideo subtitles are a gold mine of untapped potential.' %}

    • {% trans 'Captions make videos accessible for viewers who are deaf or hard of hearing ' %}
    • {% trans "Translations make it possible for all of us to watch video in languages that we don't speak" %}
    • {% trans 'Video creators get: better SEO, more views, access to a far bigger (potentially multilingual and global) audience, accessibility for deaf and hard of hearing viewers, and more' %}

  4. {% trans 'Who is the Participatory Culture Foundation and why are they making Universal Subtitles?' %}

    {% trans 'he Participatory Culture Foundation is a 501c3 non-profit organization whose mission is to build a fairer, more open, and more democratic media space. PCF saw a gap in the marketplace for more open and collaborative subtitling — Universal Subtitles is helping to fill that need.' %}

  5. {% trans 'Will Universal Subtitles support closed captions for the deaf and hard of hearing?' %}

    {% trans 'Yes, we are working on a solution for making closed captions [including information about sounds and music], as well as subtitles/translations.' %}

  6. {% trans 'Will your subtitles support my language or characterset?' %}

    {% trans 'We will be working to support as many languages/charactersets as possible.' %}

  7. {% trans 'When will the tools, including this site, be available in my language?' %}

    {% blocktrans %} The site and tools are available in a number of languages (set your language in your browser preferences and we’ll take care of the rest). We’re always looking for volunteer translators to help improve and add more languages. {% endblocktrans %}

  8. {% trans 'How can I add subtitles to my video?' %}

    {% blocktrans %} For a single video, use our site. For multiple videos on your website, use this code in your website’s template inside the <head> tags. {% endblocktrans %}

  9. {% trans 'What open source license do you use?' %}

    {% blocktrans %}For our subtitling widget and community website, we're using the AGPL.{% endblocktrans %}

  10. {% trans "I'm a developer, can I help out?" %}

    {% blocktrans %}Please do! If you're interested in coding and development, please visit our developer center.{% endblocktrans %}

  11. {% trans 'Can non-technical people get involved with the project?' %}

    {% blocktrans %} If you're interested in general involvement, please see our Mozilla Drumbeat page. If you have a specific idea, please get in touch: universalsubtitles@pculture.org And as a 501c3 non-profit org, we do accept donations (see the next question for details). {% endblocktrans %}

  12. {% trans 'Do you accept donations? Are they tax deductible?' %}

    {% blocktrans %} Yes and yes! You can donate here (through the Mozilla Drumbeat project). {% endblocktrans %}

  13. {% trans 'What is your relationship with Mozilla Drumbeat?' %}

    {% trans 'The Mozilla Foundation has provided Universal Subtitles with seed funding, community support, and fundraising support through Drumbeat. Drumbeat is a movement to keep the web open for the next 100 years, and Universal Subtitles is a featured project!' %}

  14. {% trans 'Where can I download subtitles in .SRT or other formats?' %}

    {% blocktrans %} When subtitles exist, you can download them in .SRT, .SSA, and TTML formats on the page for that language (example). Click the links in the lower right under the green "Edit Subtitles" button. {% endblocktrans %}

  15. {% blocktrans %}It says "Someone else is currently editing these subtitles" and I can't do anything. How long will this last for? Why does it happen?{% endblocktrans %}

    {% blocktrans %} The system currently only allows one user to edit subtitles for any given video at a time. So, if someone else is already editing subtitles for the video when you try to edit the subtitles, you'll get this message. {% endblocktrans %}

  16. {% trans "Why can't I change the timing on a translation?" %}

    {% blocktrans %} You can. Go to the translation editor and click “Change subtitle timing”. Only do this if the timing on the original language really cannot match the language you’re translating into. Otherwise, improvements in the original will automatically cascade. {% endblocktrans %}

  17. {% trans "How do old revisions work? What if somebody makes a change to my work that I don't approve of?" %}

    {% blocktrans %} Each set of subtitles or translation has a full revision history. Just click on the tab on the subtitle page in you will see a list of revisions. You can use the checkboxes on the left and the "compare revisions" button to see the difference between two revisions. Or you can click on any one revision, and edit based on that revision or roll back to that revision.

    If the current version is blank, but there are older revisions, it make sense to check those older revisions before starting from scratch, because there could be useful work in an older revision. {% endblocktrans %}

  18. {% trans "How do I add the Universal Subtitles "embed all videos" code to my site?" %}

    {% blocktrans %} Go to this page and double click on the code in the white box to select it all. Go to Edit > Copy in your browser's menu to copy the code to your clipboard. Then start editing the headers in your site or blog theme, and paste the script somewhere between the <head> and </head> tags in your blog theme. {% endblocktrans %}
