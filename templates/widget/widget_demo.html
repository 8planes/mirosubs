Things you can add to query string:

video_url=[video url] (NOT optional) The url of the video to embed. [video url] be a youtube page url or a video file url.
null_widget=true (optional) When true, makes the widget in null sandbox.
debug_js=true (optional) When true, opens a Javascreipt debugging panel
subtitle_immediately=true (optional) When true, opens subtitling dialog as soon as widget loads.
translate_immediately=true (optional) When true, opens translation dialog as soon as widget loads.
autoplay_language=[language code] (optional) Blank [language code] opens widget with native language subs already loaded. Or use 2-letter [language code] for other languages.
